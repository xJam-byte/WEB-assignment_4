# Analytical Platform (Backend Assignment 4)

An analytical platform that retrieves time-series data from a MongoDB database and visualizes it using interactive charts.

## Technologies Used

- **Backend**: Node.js, Express.js
- **Database**: MongoDB (with Mongoose ODM)
- **Frontend**: HTML5, CSS3, JavaScript
- **Visualization**: Chart.js
- **Utilities**: dotenv (config), cors (cross-origin requests)

## Data Source

The project uses **synthetic data** generated by the included `seed.js` script to simulate real-world scenarios:

- **Field 1**: Simulates **Temperature** (approx. 15-25Â°C) with a daily cycle pattern and random noise.
- **Field 2**: Simulates **Humidity** (30-80%) with random fluctuations.
- **Field 3**: Simulates **Traffic/Usage** (0-100) with higher values during the day and lower at night.

Measurements are generated for the last 30 days at 1-hour intervals.

## API Endpoints

### 1. Get Measurements

Fetches time-series data for a specific field within a date range.

- **URL**: `/api/measurements`
- **Method**: `GET`
- **Query Params**:
  - `field`: `field1`, `field2`, or `field3` (Required)
  - `start_date`: YYYY-MM-DD (Required)
  - `end_date`: YYYY-MM-DD (Required)
- **Response**: Array of objects containing `timestamp` and validity value.

### 2. Get Metrics

Calculates statistical metrics for a specific field.

- **URL**: `/api/measurements/metrics`
- **Method**: `GET`
- **Query Params**:
  - `field`: `field1`, `field2`, or `field3` (Required)
  - `start_date`, `end_date`: Optional filter.
- **Response**: JSON object with `average`, `min`, `max`, `stdDev`, and `count`.

## How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Configure Environment**:
    Ensure `.env` exists with your MongoDB URI (e.g., `MONGODB_URI=mongodb://localhost:27017/analytics_platform`).
3.  **Seed Database**:
    Populate the database with test data.
    ```bash
    node seed.js
    ```
4.  **Start Server**:
    ```bash
    node server.js
    ```
5.  **View Application**:
    Open `http://localhost:3000` in your browser.
